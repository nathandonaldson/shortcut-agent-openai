Shortcut Enhancement System

Welcome to the Shortcut Enhancement System API. This service automatically improves Shortcut stories based on tags.

Supported Workflows

  • Full Enhancement - Tag a story with "enhance" to analyze and automatically improve it
  • Analysis Only - Tag a story with "analyse" to get analysis without changes

API Endpoints

Webhook Receiver

POST /api/webhook/{workspace_id}

Receives webhook events from Shortcut and processes story updates with enhancement tags.

Test Pipeline

POST /api/test_pipeline

Test endpoint for running the enhancement pipeline without a webhook trigger.

{
  "workspace_id": "workspace1",
  "story_id": "12345",
  "workflow_type": "enhance" // or "analyse"
}

Setup Instructions

To use this service with Shortcut:

  1. Configure a webhook in Shortcut pointing to https://[your-domain]/api/webhook/{workspace_id}
  2. Set the webhook to trigger on story updates (specifically label changes)
  3. Tag stories with "enhance" or "analyse" to trigger the system

For more information, see the GitHub repository.
